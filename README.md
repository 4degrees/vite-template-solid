## Install

```bash
# Javascript template
$ npx degit amoutonbrady/vite-template-solid/js#main my-solid-project
$ cd my-solid-project
$ npm install # or pnpm install or yarn install
```

```bash
# Typescript template
$ npx degit amoutonbrady/vite-template-solid/ts#main my-solid-project
$ cd my-solid-project
$ npm install # or pnpm install or yarn install
```
